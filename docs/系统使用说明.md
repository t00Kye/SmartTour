# SmartTour 系统使用说明

## 一、运行环境要求

### 1.1 硬件要求

#### 开发环境
- **CPU**: Intel Core i5 或同等性能处理器及以上
- **内存**: 8GB RAM 及以上（推荐 16GB）
- **硬盘**: 至少 10GB 可用空间
- **显示器**: 分辨率 1280x800 及以上

#### 运行环境（设备）
- **设备类型**: HarmonyOS 手机
- **系统版本**: HarmonyOS 3.0 及以上
- **内存**: 4GB RAM 及以上
- **存储空间**: 至少 100MB 可用空间

### 1.2 软件要求

#### 开发工具
- **DevEco Studio**: 4.0 及以上版本
- **JDK**: JDK 1.8 或更高版本
- **Node.js**: 14.0.0 或更高版本
- **HarmonyOS SDK**: API 9 及以上

#### 运行环境
- **HarmonyOS 系统**: 3.0 及以上版本
- **网络**: 部分功能需要网络连接

## 二、环境安装步骤

### 2.1 安装 DevEco Studio

1. **下载安装包**
   - 访问 [HarmonyOS 开发者官网](https://developer.harmonyos.com/)
   - 下载对应操作系统的 DevEco Studio 安装包

2. **安装步骤**
   - Windows: 运行 `.exe` 安装程序，按提示完成安装
   - macOS: 打开 `.dmg` 文件，拖拽到应用程序文件夹
   - Linux: 解压 `.tar.gz` 文件，运行安装脚本

3. **首次启动配置**
   - 选择安装路径
   - 配置 HarmonyOS SDK 路径
   - 下载并安装必要的 SDK 组件

### 2.2 配置开发环境

1. **配置 JDK**
   ```bash
   # 检查 JDK 版本
   java -version
   ```

2. **配置 Node.js**
   ```bash
   # 检查 Node.js 版本
   node -v
   npm -v
   ```

3. **配置 HarmonyOS SDK**
   - 打开 DevEco Studio
   - 进入 `File` -> `Settings` -> `Appearance & Behavior` -> `System Settings` -> `HarmonyOS SDK`
   - 配置 SDK 路径并下载所需组件

### 2.3 导入项目

1. **打开项目**
   - 启动 DevEco Studio
   - 选择 `File` -> `Open`
   - 选择项目根目录（SmartTour 文件夹）

2. **同步依赖**
   - 等待 Gradle 同步完成
   - 等待依赖下载完成
   - 如有错误，根据提示修复

3. **配置签名（可选）**
   - 如需真机调试，需要配置应用签名
   - 进入 `File` -> `Project Structure` -> `Signing Configs`
   - 配置签名信息

## 三、系统使用说明

### 3.1 启动应用

#### 在模拟器中运行
1. 打开 DevEco Studio
2. 点击工具栏中的设备选择器
3. 选择或创建 HarmonyOS 模拟器
4. 点击运行按钮（绿色三角形）或按 `Shift + F10`
5. 等待应用安装并启动

### 3.2 功能使用指南

#### 3.2.1 首页功能

**个性化欢迎横幅**
- 显示个性化欢迎信息
- 展示推荐内容
- 点击可查看详情

**景点推荐**
- 浏览推荐的景点信息
- 查看景点图片和描述
- 点击进入详情页

**快速导航**
- 快速访问常用功能
- 包括景点、行程、商圈等入口

**智能推荐**
- 基于用户偏好推荐内容
- 个性化展示旅游信息

#### 3.2.2 景点页面

**搜索功能**
- 在搜索框输入关键词
- 支持景点名称、地点搜索
- 点击搜索按钮或回车执行搜索

**分类浏览**
- 选择景点分类（如自然风光、人文历史等）
- 浏览对应分类下的景点列表

**景点列表**
- 浏览所有景点
- 查看景点基本信息
- 点击进入详情页

#### 3.2.3 行程页面

**目的地搜索**
- 输入目的地名称
- 搜索相关旅游信息

**个性化行程推荐**
- 查看系统推荐的行程方案
- 根据个人偏好调整

**智能行程生成**
- 输入出发地、目的地、时间等信息
- 系统自动生成行程规划
- 可手动调整行程安排

#### 3.2.4 商圈页面

**商圈浏览**
- 浏览附近商圈信息
- 查看商圈内的商家

**商家推荐**
- 查看推荐的商家信息
- 查看商家评分和评价

#### 3.2.5 订单页面

**订单列表**
- 查看所有订单
- 查看订单状态（待支付、已支付、已完成等）

**订单筛选**
- 按状态筛选订单
- 按时间筛选订单

**订单详情**
- 点击订单查看详细信息
- 查看订单商品、价格等信息

#### 3.2.6 我的页面

**用户信息**
- 查看个人资料
- 编辑个人信息

**设置功能**
- 应用设置
- 账号管理

#### 页面切换

- 部分页面支持下拉刷新
